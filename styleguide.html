Styleguide for Comverse Project

The Comverse website has a few components that can be reused. These components are listed in this page. Each component has an example of HTML code and live example.

The classes all follow the SUIT CSS naming conventions, except that elements that belong to a Class--modified do not also belong to Class.

Button Links

This button would typically appear at the end of a section, and lead to a different page with more information about the same topic.

Header Buttons

This button would typically appear in a small screen device, in the page's header. The button has an icon inside it. Two different options are a hamburger icon, and a search icon, as shown below. More options can be configured by extending the SASS class %HeaderButton.

The buttons may also be wrapped with an anchor element to create links to other parts of the page.
